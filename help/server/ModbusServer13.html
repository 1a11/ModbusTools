System Settings dialog

“System Settings” – dialog shows all system settings that groups into tabs.

View

“Use Port/Device names with settings” – display settings info in ‘[]’-brackets. For port it displays port main settings, for device it displays device reference(s) (Modbus device unit address(es) ).

Log

“LogFlags” – show log message categories that was displayed in LogView;

“Use timestamp” – display timestamp for log message in LogView;

“Format” – set format for timestamp to be displayed in LogView.

Format

Result

dd.MM.yyyy

21.05.2001

ddd MMMM  d yy

Tue May 21 01

hh:mm:ss.zzz

14:13:09.120

hh:mm:ss.z

14:13:09.12

h:m:s ap

2:13:9 pm

Assumed that date and time are 21 May 2001 14:13:09.120.
